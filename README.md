# uiux-design-polish
A stack-agnostic AI coding skill that improves UI/UX quality of web interfaces.

This skill helps coding agents:

* polish visual hierarchy
* improve spacing and typography
* add missing interface states
* reduce UX friction in forms and flows
* ensure baseline accessibility (WCAG)

## When to use

Use this skill when:

* a screen looks inconsistent or hard to scan
* forms cause user errors
* loading / empty / error states are missing
* accessibility issues need to be fixed
* doing a final UI/UX QA pass before release

## Key capabilities

* UI hierarchy improvements
* spacing and typography normalization
* component state coverage
* accessibility baseline checks
* perceived performance improvements
* UI QA validation workflow

## Example triggers

* "Improve the UI of this page, spacing and hierarchy are inconsistent."
* "Audit this form for UX and accessibility problems."
* "Polish this dashboard layout."
